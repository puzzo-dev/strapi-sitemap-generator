# Ad Images Directory

This directory contains images for dynamic ads displayed throughout the I-Varse website.

## Required Images

### Blog Sidebar Ads
- `erp-dashboard.jpg` - ERP solutions ad showing a dashboard interface
- `cloud-infrastructure.jpg` - Cloud services ad showing server/cloud infrastructure
- `custom-development.jpg` - Custom development ad showing code or development workspace

### Campaign Ads
- `year-end-promo.jpg` - Year-end promotional ad with festive/business theme

### Industry-Specific Ads
- `healthcare-system.jpg` - Healthcare IT solutions showing medical technology
- `fintech-platform.jpg` - FinTech solutions showing financial/banking interface

### Performance Ads
- `success-metrics.jpg` - Popular choice ad showing charts, metrics, or success indicators

## Image Specifications
- **Format**: JPG or PNG
- **Dimensions**: 400x200px (2:1 aspect ratio recommended)
- **Optimization**: Images should be optimized for web (under 100KB each)
- **Style**: Professional, modern, aligned with I-Varse branding

## Usage
These images are displayed as background overlays in the DynamicAdCarousel component with 20% opacity, so they should be clear and not overly complex to maintain text readability.
